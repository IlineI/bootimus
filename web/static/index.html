启动菜单主题

自定义客户端看到的 PXE 启动菜单外观。 并非所有 iPXE 固件版本都支持控制台分辨率。

自动启动默认条目前的等待时间。设置为 0 以禁用(永久等待)。

Webhook

当 PXE 事件发生时,将 JSON 负载 POST 到 URL。发送后不管,10秒超时,不重试。

备份

下载 bootimus 数据目录的 .tar.gz 备份:数据库、引导加载程序配置、自定义引导加载程序集和设置。不包括 ISO 和下载的工具二进制文件(它们可以重新生成)。

恢复是手动的:停止 bootimus,将 tarball 解压到数据目录上,启动 bootimus。

USB 启动镜像

为不支持 PXE 网络启动的机器下载可启动的 USB 镜像。 写入 U 盘并从中启动以访问 bootimus 菜单。
